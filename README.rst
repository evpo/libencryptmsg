libEncryptMsg
=============

This is an OpenPGP implementation that works with symmetrically
encrypted files following RFC4880 specification.

CLI:

::

    ./bin/debug/encryptmsg

Unit tests:

::

    ./bin/debug/encryptmsg-tests

System requirements
-------------------

::

    Linux
    Mingw
    C++11 g++ or clang

Compile
-------

Build:

::

    make

Release:

::

    make RELEASE=on

Verbose:

::

    make VERBOSE=on

Clean:

::

    make clean

Tests:

::

    make tests
